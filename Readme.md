# Sistema de Gestión de Estacionamiento

## 📌 Descripción

Sistema desarrollado en Java para la administración de estacionamientos. Permite gestionar clientes, vehículos, cajones, pensiones, precios y reportes.

---

## ⚙️ Tecnologías

* Java
* Swing
* Maven
* JDBC

---

## 🚀 Ejecución

```bash
java -jar target/ppark-1.0.0.jar
```

O desde IDE ejecutando la clase `Main`.

---

## 🗄️ Configuración

Editar el archivo:

```
config.properties
```
